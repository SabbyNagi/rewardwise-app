# RewardWise

**We optimize your wallet. You just travel.**

AI-powered travel rewards optimization platform that delivers one verdict — use points, pay cash, or wait.

## 🚀 Live Demo

[https://rewardwise-app.vercel.app](https://rewardwise-app.vercel.app)

## 🛠️ Tech Stack

- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🎯 Features (MVP)

- **The Verdict:** Single recommendation (points vs. cash)
- **Zoe AI Assistant:** Natural language search input
- **Savings Calculator:** Real-time value comparison
- **Transfer Path Optimizer:** Best points transfer routes
